Introduction

HTML is the standard markup language for creating web pages. It provides a structure for the content of a web page, including text, images, links, and other elements. HTML uses a system of tags and attributes to define the structure and appearance of the content.

HTML documents consists a hierarchy of elements, which can be nested inside each other. Each element has a specific purpose and contain other elements or text. Elements can also have attributes, which provide additional information or functionality.

  • HTML elements are represented by tags, which are enclosed in angle brackets (). For example < p > represent a pragraph element.
  • HTML attributes provide additional information about an element, and are specified within the opening tag of the element. For example, the href attribute in an tag specifies the URL of the link

HTML and CSS

HTML and CSS work together to create web pages. HTML defines the structure and content of the page, while CSS controls the presentation and styling of the elements. By combining HTML and CSS you can create visually appealing and interactive web pages.

  • HTML provides a set of elements and attributes that define the structure of the page, such as headings, paragraphs, lists, images, and links. CSS, on the other hand, allows you to control the appearance of these elements, such as their colors, fonts, sizes, and layouts.

  • By using CSS, you can apply styles to HTML elements using selectors and declarations. Selectors target specific elements, while declarations define the styles to be applied. CSS can be included in an HTML document using the style tag, or it can be stored in separate CSS file and linked to the HTML document using the link tag.

Elements

Html document are made up of elements, which represents different types of content. Each element has a specific purpose and can contain other elements or text. Elements are represented by tags, which are enclosed in angle brackets ( < and > ). The opening tag indicates the start of an element, while the closing tag indicates the end of the element.

Elements ca be nested inside each other, creating a hierarchy of elements. The outermost element is the < html > element, which contains all other elements in the document. Elements can also have attributes, which provide additional information or functionality. Attributes are specified within the opening tag of the element.

Element Hierarchy

HTML elements can be nested inside each other, creating a hierarchy of elements. The outermost element is the < html > element, which contains all other elements in the document. The < body > element represents the main content of the document, while the < head > element contains meta information about the document and external resources such as stylesheets and scripts.

Elements can be nested inside other elements to create more complex structures. For example, a < div > element can contain multiple other elements, such as headings, paragraphs, and images. This nesting allows you to create structured and organized content.

It is important to properly close each element by using the corresponding closing tag. Failure to do so can result in unexpected rendering of the document.

Global Attributes

HTML elements can have attributes, which provide additional information or functionality to the elements. Some attributes are specific to certain elements, while others can be used with any element. These common attributes are known as global attributes.

Some commonly used global attributes inlude:

  • class: Specifies one or more class names for an element

  • id: Specifies a unique id for an element

  • style: Specifies inline CSS style for an element

  • title: Specifies extra information about ab element
